# Sync Vision Command

Add a `method sync vision` command that runs the Executive Summary
Protocol and refreshes `docs/VISION.md` as a bounded, provenance-carrying
signpost. It should inventory the governing repo surfaces, generate the
summary from repo truth, and attach a witness reference or manifest so a
later human or agent can inspect how the summary was derived.

This idea already surfaced in spirit once `method sync ship` automated
other signpost maintenance. A dedicated vision sync would make summary
refresh cheaper, more reproducible, and more agent-friendly than
manually re-running a large read/summarize pass with ad hoc prompts.
