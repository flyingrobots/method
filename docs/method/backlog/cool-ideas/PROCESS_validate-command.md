# Validate Command

Add a `method validate` command that checks repo-visible METHOD
invariants across backlog, design, retro, legends, and generated
signposts. This should be more than a general doctor command: it should
make the documentation and metadata contract cheap to verify for humans,
CI, and agents before drift leaks into committed state.
