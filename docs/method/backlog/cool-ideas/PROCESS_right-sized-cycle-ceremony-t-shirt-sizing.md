# Right-sized cycle ceremony (t-shirt sizing)

METHOD treats every unit of work the same way — a one-line fix and a major refactor both require 8-section design docs, playback questions, posture sections, and 6-section retros. Add t-shirt sizing: S (commit + test, no design doc), M (light design doc + tests + brief retro), L (full cycle with all postures). The current default of 'everything is L' is the primary source of ceremony-to-value friction.
