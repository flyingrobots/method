# Meaningful retros over template retros

Almost every retro written this session has 'Drift: None recorded. New Debt: None recorded. Cool Ideas: None recorded.' The 6-section template creates the shape of reflection without producing it. Replace with two focused prompts: 'What surprised you?' and 'What would you do differently?' Two honest sentences beat five empty sections.
