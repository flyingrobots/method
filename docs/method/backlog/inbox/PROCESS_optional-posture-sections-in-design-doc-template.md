# Optional posture sections in design doc template

The design doc template requires accessibility, localization, and agent inspectability posture sections even for trivial changes where the answer is always 'not in scope.' These should be required for L-sized cycles and available-but-not-mandatory for M-sized ones. The pull scaffold should ask or infer the cycle size and emit the appropriate template.
